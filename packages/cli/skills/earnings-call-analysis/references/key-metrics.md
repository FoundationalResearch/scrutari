# Key Metrics for Earnings Call Analysis

## Universal Metrics (All Companies)

| Metric | What to Look For |
|--------|-----------------|
| Revenue | YoY growth rate, beat/miss vs consensus |
| EPS (GAAP) | Actual vs estimate, YoY change |
| EPS (Non-GAAP) | Adjustments made, actual vs estimate |
| Gross Margin | Trend direction, pricing vs cost drivers |
| Operating Margin | Leverage improvement or deterioration |
| Free Cash Flow | Conversion rate vs net income |
| Guidance (Revenue) | Range midpoint vs consensus, width of range |
| Guidance (EPS) | Implied margin trajectory |

## Sector-Specific KPIs

### Technology / SaaS
- Annual Recurring Revenue (ARR)
- Net Revenue Retention (NRR)
- Customer count and average deal size
- Rule of 40 (revenue growth % + FCF margin %)
- Remaining Performance Obligations (RPO)

### Consumer / Retail
- Same-store sales (comps)
- Average order value (AOV)
- Customer acquisition cost (CAC)
- Store count changes
- Inventory turnover

### Financial Services
- Net Interest Margin (NIM)
- Provision for credit losses
- Return on Equity (ROE)
- Efficiency ratio
- Assets under management (AUM)

### Healthcare / Pharma
- Pipeline updates (phase transitions)
- Prescription volumes
- Payer mix changes
- R&D spend as % of revenue
- Patent expiration timeline

### Industrials / Manufacturing
- Book-to-bill ratio
- Backlog growth
- Capacity utilization
- Price vs volume decomposition
- Raw material cost trends

## Guidance Analysis Framework

| Signal | Interpretation |
|--------|---------------|
| Raised full-year, maintained next Q | Confidence building, conservative near-term |
| Raised both | Strong momentum, high confidence |
| Maintained both | In-line execution, no surprises |
| Lowered next Q, maintained full-year | Near-term headwind, expects recovery |
| Lowered both | Deteriorating fundamentals, high concern |
| Narrowed range | Increased visibility, less uncertainty |
| Widened range | Increased uncertainty, macro concerns |
| Withdrew guidance | Significant uncertainty, red flag |
